## 步骤4：材料掉落+锻造

### 4.1 材料定义
- 🪨 铁锭（基础）、⚙️ 精钢锭（中阶）
- 🔥 元素精华、⚔️ 战魂精华、🛡️ 守护精华
- 👑 地精王冠碎片（Boss专属）
- 材质最多5个，精华总量最多8个，Boss材料最多1个

### 4.2 材料掉落
- 战后二选一：[卡牌三选一] 或 [材料奖励]
- 普通：铁锭×1 / 精英：精钢锭×1+随机精华×1
- Boss：额外给材料（不参与二选一）

### 4.3 铁匠节点
- ⚒️ 铁匠，第一幕1个
- 可锻造武器：
  - 铁制长剑(铁锭×2)：普攻6伤，战技后下张-1费
  - 铁制匕首(铁锭×2)：普攻3×2伤，首张≤1费额外抽牌
  - 铁制战锤(铁锭×2)：普攻10伤，单次≥15伤击碎3甲
  - 铁制弓(铁锭×2)：普攻5伤，未受伤回合+30%伤害
  - 铁制法杖(铁锭×2)：普攻3伤，法术+20%，每出法术+1蓄能
  - 精钢长剑(精钢锭×2+精华×1)：普攻8伤，同铁制特性

### 4.4 装备系统
- 主手武器1个槽位
- 装备后新增"普攻"(0费，每回合限1次)
- 武器特性自动生效

### 4.5 材料应急使用（战斗中消耗，每种每场限1）
- 🪨 铁锭：+8护甲
- ⚙️ 精钢锭：+12护甲
- 🔥 元素精华：全体敌人+2灼烧
- ⚔️ 战魂精华：本场力量+2
- 🛡️ 守护精华：本场每回合自动+3护甲
- 使用前需确认提示

- [x] 战后二选一正常（卡牌vs材料）
- [x] Boss战同时给卡牌和材料
- [x] 材料正确显示和累计，不超上限
- [x] 铁匠可进入，材料够时能锻造
- [x] 装备武器后出现普攻选项，每回合限1次
- [x] 武器特性正确生效
- [x] 材料战斗中应急使用正常，有确认提示

## 步骤5：附魔+共鸣

### 5.1 附魔台节点
- 🔮 附魔台，第一幕1个（和铁匠不同位置）
- 每件武器最多2个附魔槽，可覆盖不可拆除

### 5.2 附魔效果（各消耗元素精华×1）
- 🔥 烈焰：普攻/战技命中施加1灼烧
- ❄️ 寒冰：普攻/战技命中20%冻结
- ⚡ 雷电：普攻/战技命中弹射3伤害到随机另一敌人
- 💀 汲魂：击杀回复5HP
- 🌀 虚空：普攻/战技无视3点护甲
- ✨ 祝福：战技伤害+3

### 5.3 共鸣系统（双附魔组合触发）
- 🔥+❄️ "熔岩"：灼烧伤害翻倍
- 🔥+⚡ "雷焰"：弹射也施加1灼烧
- ❄️+⚡ "风暴"：冻结解除时造成15伤害
- 💀+🌀 "死神"：击杀回复全额HP
- 🔥+✨ "圣火"：对有灼烧目标额外+50%伤害

### 5.4 附魔UI
- 上方当前武器+已有附魔 / 中间可选附魔列表 / 下方确认
- 2槽满时选择覆盖哪个
- 可触发共鸣时高亮提示"✨可触发共鸣"
- 战斗中附魔触发有视觉反馈
