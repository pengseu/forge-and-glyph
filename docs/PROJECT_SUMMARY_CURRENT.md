# 《锻铸与咒印》当前实现总结（Phase 3 进行中）

> 更新时间：当前 `phase3-step6` 工作分支状态

## 1. 游戏定位
- 类型：网页端回合制卡牌 Roguelike（构筑 + 地图路线）
- 核心循环：
  1. 地图选点
  2. 战斗
  3. 战后奖励（卡牌/材料）
  4. 节点功能（篝火/商店/铁匠/附魔/事件）
  5. 前往 Boss 结算

## 2. 当前可玩流程
- 第一幕为 7 层分叉地图（含普通战斗、精英、事件、篝火、Boss）
- 一局可完整打到 Boss 并胜负结算
- 地图支持滚动与拖拽浏览（可点到底层节点）

## 3. 战斗系统
### 3.1 资源与回合
- 双资源：体力（Stamina）/魔力（Mana）
- 支持 `free`（免费）与 `hybrid`（双资源）费用卡
- 回合制：玩家阶段 -> 敌人阶段 -> 结算 -> 下一回合

### 3.2 状态效果
- 已实现：灼烧、冻结、中毒、力量、虚弱、易伤、屏障、智慧、蓄能
- 结算规则已接入并有测试覆盖（含 DOT、衰减、免疫等）

### 3.3 武器与普攻
- 装备后可用普攻（每回合限一次）
- 已实现多种武器行为（长剑/匕首/战锤/弓/法杖系效果）

### 3.4 附魔与共鸣
- 6 种附魔：flame/frost/thunder/soul/void/bless
- 双槽附魔，支持覆盖
- 已接入多组共鸣（如雷焰、圣火、熔岩、风暴、死神）

### 3.5 材料战斗内应急使用
- 材料可在战斗中触发一次性效果（每种每场限一次）

## 4. 卡牌系统
### 4.1 卡池
- 当前卡池已扩展到 40 张
- 包含基础/普通/稀有/史诗分层

### 4.2 Step6 目标新增卡已接入
- 快攻、双重打击、引燃、寒霜新星、蚀骨毒、易伤诅咒
- 透支、魔力涌流、平衡、荆棘甲、魔法吸收
- 剑魔合一、血之狂怒

### 4.3 升级
- 篝火支持卡牌升级
- 新卡升级映射已补齐

## 5. 地图与节点
### 5.1 地图结构
- 7 层路线，Boss 固定收束
- 含 2 精英、2 篝火、商店、铁匠、附魔台、事件节点

### 5.2 节点类型
- `normal_battle` 普通战斗
- `elite_battle` 精英战斗
- `boss_battle` Boss 战
- `campfire` 篝火
- `shop` 商店
- `forge` 铁匠
- `enchant` 附魔台
- `event` 未知事件

## 6. 事件系统（❓）
- 已接入 5 类事件及权重抽取：
  - 神秘商人
  - 废弃营地
  - 祭坛
  - 旅行者
  - 锻造之灵
- 事件结果可写回 run 状态（HP/金币/材料/卡牌/升级）
- 事件可触发额外战斗

## 7. 经济与奖励
- 战斗金币奖励按节点类型区分（普通/精英/Boss）
- 商店支持买卡、回血、移除卡牌
- 战后奖励支持卡牌与材料路径

## 8. 结算页与调优信息
- 已支持基础结算（胜负、回合、剩余 HP）
- 已接入结构化 run 报告：
  - 路线
  - 每场战斗日志
  - 全局操作日志
  - 终局背包快照（武器/材料/金币/卡组数）

## 9. 当前玩法体验（已验证）
- 可完整游玩一局并产生构筑方向
- Boss 战已从“过于简单”向“需策略出牌”方向调整
- 篝火已限制为每次节点单次操作后离开（修复可重复升级问题）

## 10. 已知问题 / 下一步重点
- 事件 `altar_blood` 选择 Epic 时出现高重复（需优先修）
- 结算页统计维度仍可增强：
  - 总回合语义拆分（整局 vs 单战）
  - 失败原因摘要
  - 经济/构筑统计聚合
  - 日志分层展示（摘要/详细）
- 目标时长仍偏短，需继续平衡到 15-25 分钟区间
