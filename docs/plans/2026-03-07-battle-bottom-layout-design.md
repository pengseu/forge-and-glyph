# 战斗底部操作区重构设计

日期：2026-03-07  
范围：战斗场景底部操作区与手牌布局，不改战斗逻辑与数值。

## 背景

当前战斗页底部独立材料栏占据了一整行高度，压缩了战斗主舞台的可用纵向空间；同时手牌仍采用扇形展开，不再贴合新的手牌盒视觉。

## 目标

- 移除独立材料栏，把空间还给战斗区
- 在手牌盒左侧新增固定宽度的森林采集袋
- 有材料时直接显示可点击小徽章；空状态为氛围型空袋插画
- 底部操作区改为三段式：材料袋 / 手牌盒 / 操作按钮区
- 手牌改为平铺主布局，主承载宽度 600px，超出时轻微叠放
- 战斗主舞台相对当前布局整体下移，增强角色落地感

## 方案

1. 删除 `.battle-material-bar` 作为独立块级区域
2. 把材料交互合并进 `.hand-area` 左侧的 `.materials-pocket`
3. `.hand-area` 内部采用三列：左袋、中牌、右操作
4. 手牌从扇形算法切换为平铺/叠放算法，不再大角度旋转
5. 把抽牌堆/弃牌堆与普攻/结束回合收纳到右侧 `action-cluster`
6. 战斗舞台整体下沉一档，使中景画面更接近参考构图

## 验收

- 页面不再出现独立材料横条
- 材料袋固定宽度，空状态不突兀
- 手牌在 600px 范围内平铺，超出后轻微叠放
- 战斗层获得更多垂直空间，敌我更贴地
- 不影响出牌、选目标、用材料、结束回合等交互
