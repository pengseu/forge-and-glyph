# UI 素材化现状总表（2026-03-08）

## 目标

梳理当前主要页面中，哪些元素已经完成素材化，哪些仍然使用文字占位或纯文字表达，便于后续继续统一视觉语言时快速定位。

## 判定规则

- 已素材化：页面直接引用图片素材，且为实际玩家可见的重要视觉元素
- 文字占位：当前仍靠单字、短字或纯文字承担原本更适合素材化的视觉角色
- 纯文字保留：功能上更清晰，短期内不建议为了“去文字”而再素材化

## 建议优先级

### P1：建议下一批继续处理

1. `附魔页` 的附魔槽位单字，占位意义明显，后续适合换成小型附魔徽记
2. `战斗页 / 商店页` 的卡牌费用角标，当前功能清楚，但视觉统一度还可以继续提升
3. `地图页` 顶部武器栏的兜底短字，仅在缺少武器缩略图时出现，后续可继续补图减少回退场景

### P2：建议保持文字，不要过度素材化

1. 标题、标签页、普通按钮文案
2. 金币、牌组、幕数这类短信息标签
3. “关闭”“已升级”“立即装备”这类动作性或状态性文案

## 本轮结论

当前项目里，真正影响统一性的文本 `emoji` 已经清空；剩余需要关注的，主要不是 `emoji`，而是少量“文字型功能占位”。这些占位里，最值得下一步处理的是：

- 附魔槽位单字
- 卡牌费用角标
- 地图顶部武器缺图兜底短字
