# 《锻铸与咒印》开发进度

## 已完成功能

### Phase 1 — 最小可玩（已完成）
- 双资源战斗核心（体力 3 / 魔力 2，每回合刷新）
- 10 张基础卡牌（战技、法术、技艺三类）
- 1 种敌人（地精散兵）
- 战斗 UI（玩家状态栏、敌人区域、手牌区域）
- 回合制战斗流程（抽牌、出牌、结束回合、敌人行动）
- 灼烧状态效果
- 胜利/失败结算

### Phase 2 — Roguelike 感（已完成）
- 路线图系统（9 节点：5 普通 + 2 精英 + 1 篝火 + 1 Boss）
- 分支路径，玩家可选路线
- 战后选卡（三选一或跳过）
- 4 种敌人：地精散兵、森林狼、蘑菇怪、地精王（Boss）
- 20 张卡牌（10 张新增：冻结箭、毒液喷射、力量咒文、虚弱诅咒、猛击、铁壁、陨石术、血魔法、狂暴、奥术灌注）
- 卡组跨战斗持久化
- HP 跨战斗持久化

### Phase 2 补充 — 篝火 + 装备系统（已完成）
- 篝火节点（路线图中固定 1 个）
  - 回血（恢复到满血）
  - 升级卡牌（选择伤害+2 或费用-1，每张卡最多升级 1 次）
- 装备系统
  - 长剑：战技卡打出后下一张战技费用-1
  - 精钢长剑：战技卡打出后下一张战技费用-2
  - 战斗胜利后 30% 概率掉落长剑
  - 篝火可升级长剑为精钢长剑
- 完整状态效果系统
  - 🔥 灼烧：回合初受 N 伤害，层数-1
  - 🧊 冻结：跳过敌人下一次行动
  - 🐍 中毒：回合初受 N 伤害，层数不递减
  - 💪 力量：物理伤害+N
  - 😵 虚弱：敌人伤害-25%
  - ✨ 奥术灌注：下一张战技卡伤害+50%
  - 狂暴：消耗所有魔力转为体力
- 战斗 UI 显示装备武器、折扣费用、敌人状态效果

---

## 技术栈
- Vite + TypeScript + Vitest
- 纯 DOM 渲染，无框架
- 像素复古风 UI（Press Start 2P 字体）
- 48 个测试全部通过

## 项目结构
```
src/
  game/
    types.ts        — 所有类型定义
    cards.ts        — 20 张卡牌定义
    enemies.ts      — 4 种敌人定义
    weapons.ts      — 武器定义（长剑、精钢长剑）
    combat.ts       — 战斗系统（出牌、回合、武器效果）
    effects.ts      — 卡牌效果应用（所有状态效果）
    campfire.ts     — 篝火系统（回血、升级卡牌）
    map.ts          — 路线图生成（9 节点）
    run.ts          — 冒险状态管理（卡组、装备、节点）
    reward.ts       — 战后奖励卡牌
    __tests__/      — 7 个测试文件
  ui/
    renderer.ts     — 场景路由
    scenes/
      title.ts      — 标题场景
      map.ts        — 路线图场景
      battle.ts     — 战斗场景
      campfire.ts   — 篝火场景
      reward.ts     — 选卡场景
      result.ts     — 结算场景
  main.ts           — 游戏控制器
  style.css         — 全局样式
```
