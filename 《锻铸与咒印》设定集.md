# 《锻铸与咒印》(Forge & Glyph) 游戏设计文档

## 一、项目概述



* **类型**：Web 端 Roguelike 卡牌构筑游戏（HTML + CSS + JS 单页面）

* **世界观**：剑与魔法 —— 巨兽、巨龙、地精、武器锻造与附魔

* **核心一句话**：双资源驱动的卡牌战斗 × 局内武器锻造与附魔 × Roguelike 地牢探索

* **参考游戏**：Slay the Spire（卡牌构筑）+ Hades（Meta 进度思路）+ Into the Breach（信息透明策略感）

### 与 Slay the Spire 的核心差异



| Slay the Spire | 锻铸与咒印            |
| -------------- | ---------------- |
| 单一能量资源         | 体力 + 魔力双资源博弈     |
| 卡牌是唯一构筑维度      | 卡牌 + 装备 + 附魔三层构筑 |
| 遗物被动触发         | 装备主动影响卡牌效果       |
| 角色决定卡池         | 武器类型决定战斗风格       |

## 二、核心战斗系统

### 2.1 双资源：体力 & 魔力



* 每回合刷新：体力恢复至上限（默认 3），魔力恢复至上限（默认 2），抽 5 张牌

* 体力驱动物理系卡牌（挥砍、格挡、冲锋、锻造……）

* 魔力驱动魔法系卡牌（火球、护盾、附魔、召唤……）

* 每回合必须在两种资源之间做取舍，不能 "随便花"

### 2.2 资源桥接卡

部分卡牌可以打破资源边界，是后期 build 的核心引擎：



* 体力→魔力转化（如 "战吼"）

* 魔力→体力转化（如 "冥想"）

* HP→魔力转化（如 "血魔法"）

* 全资源倾斜（如 "狂暴"：消耗所有魔力转为体力）

* 跨系增幅（如 "奥术灌注"：魔力增幅下一张物理卡伤害）

### 2.3 三类卡牌



| 类别     | 消耗 | 代表                       |
| ------ | -- | ------------------------ |
| ⚔️ 战技卡 | 体力 | 挥砍、格挡、重斩、连击、处决、旋风斩       |
| ✨ 法术卡  | 魔力 | 火花、火球术、寒冰箭、连锁闪电、陨石术、灵魂虹吸 |
| 🔧 技艺卡 | 任意 | 临时锻造、淬毒、附魔、分解、陷阱、修复护甲    |

**稀有度**：分四级，⚪基础 → 🟢普通 → 🔵稀有 → 🟣史诗

### 2.4 状态效果



| 效果     | 说明                 |
| ------ | ------------------ |
| 🔥 灼烧  | 回合末受 N 伤害，层数 - 1   |
| 🧊 冻结  | 跳过下一次行动            |
| 🐍 中毒  | 回合末受 N 伤害，层数不自然消退  |
| 💪 力量  | 物理伤害 + N           |
| 🔮 智慧  | 法术伤害 + N           |
| 🛡️ 护甲 | 抵消等量伤害，回合末清零       |
| 🔵 屏障  | 护甲回合末不清零（最多保留 N 层） |
| ⚡ 蓄能   | 每层增加下一个法术 10% 伤害   |
| 💀 易伤  | 受到伤害 + 50%         |
| 😵 虚弱  | 造成伤害 - 25%         |

## 三、装备系统（第二层构筑）

### 3.1 装备栏位

主手武器 / 副手 / 护甲 / 饰品 ×2

### 3.2 武器类型（决定战斗风格）



| 武器     | 普攻    | 特性                  | 适合流派     |
| ------ | ----- | ------------------- | -------- |
| ⚔️ 长剑  | 6 伤害  | 战技卡打出后下一张战技费 - 1    | 战技流、连击流  |
| 🗡️ 匕首 | 3×2   | 首张卡费≤1 时额外抽一张       | 低费海、毒刃流  |
| 🔨 战锤  | 10 伤害 | 单次≥15 伤害击碎 3 护甲     | 大伤害、处决流  |
| 🏹 弓   | 5 伤害  | 未受伤回合伤害 + 30%       | 控制流、风筝流  |
| 🪄 法杖  | 3 伤害  | 法术 + 20%，每出法术获 1 蓄能 | 纯法流、法术爆发 |

### 3.3 武器获取



* 掉落：地牢中直接获得（随机词缀）

* 锻造：篝火 / 铁匠事件中用材料打造

* Boss 掉落：传说级武器

## 四、附魔系统（第三层构筑）

### 4.1 规则



* 来源：附魔台事件 / 技艺卡 / 商人

* 每件武器最多 2 个附魔槽

* 附魔可覆盖不可拆除

### 4.2 附魔与共鸣



| 附魔    | 效果        | 共鸣组合                    |
| ----- | --------- | ----------------------- |
| 🔥 烈焰 | 攻击施加 1 灼烧 | + 寒冰 ="熔岩"：灼烧伤害 ×2      |
| ❄️ 寒冰 | 20% 概率冻结  | + 雷电 ="风暴"：冻结解冻时 15 伤害  |
| ⚡ 雷电  | 弹射 3 伤害   | + 烈焰 ="雷焰"：弹射也施加灼烧      |
| 💀 汲魂 | 击杀回 5HP   | + 虚空 ="死神"：击杀回全额 HP     |
| 🌀 虚空 | 无视 3 护甲   | —                       |
| ✨ 祝福  | 战技 + 3    | + 烈焰 ="圣火"：灼烧目标额外 + 50% |

**核心**：共鸣是玩家追求特定附魔组合的核心动力。

## 五、锻造系统（跨局设计）

**核心原则**：带不走剑，但带得走手艺

### 锻造层级



| 层级     | 内容                   | 持久性                 |
| ------ | -------------------- | ------------------- |
| 局外・知识层 | 蓝图、附魔谱、锻造技法、矿脉图      | 永久解锁 —— 扩展可能性，不增加数值 |
| 局内・实体层 | 具体武器、材料库存、附魔实例       | 死亡全部清零 —— 每局核心构筑乐趣  |
| 传承・连接层 | 通关后传承 1 件武器到下局（弱化出现） | 跨局桥梁 —— 情感联结 + 长期叙事 |

### 蓝图解锁示例



* 首次用长剑击败地精王 → 解锁 "精钢长剑" 蓝图（以后每局铁匠可造）

* 首次用火焰附魔杀冰 Boss → 解锁 "焚尽" 附魔谱（烈焰升级版加入池）

* 首次集齐 5 种矿石 → 解锁 "合金术" 技法（混合材质锻造）

**设计特点**：新手铁匠处只有 3 种基础武器，老玩家同一个铁匠处可能有几十种选择 —— 数值差距小，路线选择丰富。

### 传承系统



* 条件：通关（击败最终 Boss）后可选 1 件武器传承

* 效果：下一局中后期出现事件获得该武器弱化版（附魔保留 1 个 / 词缀降级）

* 传承 3 次以上的武器获得 "传说" 前缀并记录历史战绩

* 死亡不可传承，只有通关才行

### 补充一：Boss 传奇武器与蓝图系统

#### 传奇武器获取

每个 Boss 固定掉落一把传奇武器，当局直接装备使用。

#### 传奇 vs 蓝图锻造版



| 维度   | Boss 掉落传奇原版 | 蓝图锻造凡人版                        |
| ---- | ----------- | ------------------------------ |
| 数值   | 基础属性高       | 基础属性约为原版 70%                   |
| 特效   | 完整特效        | 弱化版特效                          |
| 附魔槽  | 3 个（传奇专属）   | 2 个（标准）                        |
| 获取方式 | 仅 Boss 掉落   | 解锁蓝图后每局铁匠可锻造                   |
| 锻造成本 | 免费直接获得      | 需要对应材质 ×2 + Boss 材料 ×1 + 精华 ×1 |

**设计特点**：两者 build 路线不同：传奇原版自带满特效不需要附魔，蓝图版特效弱但可以通过附魔走出不同方向。

#### 三个 Boss 传奇武器



| Boss | 传奇武器      | 传奇效果                  | 蓝图凡人版效果                 | 锻造材料                      |
| ---- | --------- | --------------------- | ----------------------- | ------------------------- |
| 地精王  | 🗡️「蚁群」匕首 | 每次攻击召唤地精幽灵协同攻击 (3 伤害) | 每第 3 次攻击召唤一只地精幽灵 (3 伤害) | 铁锭 ×2 + 地精王冠碎片 ×1 + 精华 ×1 |
| 熔岩巨人 | 🔨「碎星」战锤  | 击杀时爆炸，对全体造成溢出伤害       | 击杀时对相邻敌人造成 5 伤害         | 秘银 ×2 + 熔岩核心 ×1 + 精华 ×1   |
| 远古巨龙 | 🪄「龙言」法杖  | 每打出法术卡随机施放额外法术效果      | 每打出第 3 张法术卡随机施放额外法术     | 龙骨 ×2 + 龙心碎片 ×1 + 精华 ×1   |

#### 蓝图精通等级

蓝图可通过反复使用升级，但永远略弱于传奇原版：



| 精通等级  | 解锁条件          | 效果                    |
| ----- | ------------- | --------------------- |
| 0（解锁） | 首次获得传奇原版      | 可锻造基础凡人版              |
| 1     | 用凡人版击杀 10 个精英 | 锻造时可选一个额外词缀           |
| 2     | 用凡人版通关一次      | 凡人版特效数值提升             |
| 3     | 用凡人版完成一次传承    | 解锁第 3 附魔槽，名称变为「XX・传承」 |

#### 传奇武器与传承系统的关系



* 传奇原版可以传承（通关后选择传承 → 下局中后期以弱化版出现）

* 蓝图凡人版也可以传承（同样规则）

* 蓝图精通等级 3 要求用凡人版完成传承，形成长期追求目标

#### 彩蛋：真传奇

若某局同时持有传奇原版 + 蓝图凡人版（需要当局打 Boss 掉落原版，同时之前已解锁蓝图并在铁匠锻造了凡人版），触发隐藏事件 "两把武器产生共鸣"，合并为「XX・真」：拥有原版全部效果 + 3 附魔槽 + 独特外观，局末自动传承（无需通关）。

### 补充二：材料系统

#### 设计原则

每局材料够锻造 1-2 件装备，但无法锻造所有想要的装备。材料不是 "够不够" 的问题，而是 "够造哪一把" 的抉择。

#### 材料分类

总共 10 种材料，分三大类：



1. **材质（4 种，决定武器品质）**

   \| 材质 | 定位 | 对应武器基础属性 |

   \| ---- | ---- | ---- |

   |🪨 铁锭 | 基础，到处可得 | 100%|

   |⚙️ 精钢锭 | 中阶，中期出现 | 130%|

   |💎 秘银锭 | 稀有，后期为主 | 160%|

   |☄️ 陨铁锭 | 极稀有，需局外解锁蓝图 | 200%|

2. **精华（3 种，通用货币，不细分元素类型）**

   \| 精华 | 用途 |

   \| ---- | ---- |

   |🔥 元素精华 | 附魔消耗 |

   |⚔️ 战魂精华 | 武器词缀消耗 |

   |🛡️ 守护精华 | 护甲 / 饰品升级消耗 |

3. **Boss 材料（3 种，专属掉落，每种仅 1 个）**

   \| 材料 | 来源 | 用途 |

   \| ---- | ---- | ---- |

   |👑 地精王冠碎片 | 地精王掉落 | 锻造「蚁群・仿」|

   |🌋 熔岩核心 | 熔岩巨人掉落 | 锻造「碎星・仿」|

   |🐉 龙心碎片 | 远古巨龙掉落 | 锻造「龙言・仿」|

#### 锻造成本



| 武器品质    | 材质消耗    | 精华消耗                 |
| ------- | ------- | -------------------- |
| 铁武器     | 对应材质 ×2 | 无                    |
| 精钢武器    | 对应材质 ×2 | 任意精华 ×1              |
| 秘银武器    | 对应材质 ×3 | 任意精华 ×2              |
| 陨铁武器    | 对应材质 ×3 | 任意精华 ×3              |
| Boss 仿品 | 对应材质 ×2 | Boss 材料 ×1 + 任意精华 ×1 |

#### 掉落经济

##### 第一幕（边境荒野）预期收获

铁锭 4-5 / 精钢锭 1-2 / 精华 1-2 / Boss 材料 1



* 普通战斗：每场掉 1 铁锭（100%）

* 精英战斗：掉 1 精钢锭 + 1 精华

* Boss：掉 Boss 材料 ×1 + 精钢锭 ×1 + 精华 ×1

* 够造 1 把铁武器，或攒精钢留到第二幕

##### 第二幕（深渊矿洞）预期收获

铁锭 2-3 / 精钢锭 2-3 / 秘银锭 1-2 / 精华 3-4 / Boss 材料 1



* 普通战斗：掉 铁锭 (50%) 或 精钢锭 (50%)

* 精英战斗：掉 1 秘银锭 + 1-2 精华

* Boss：掉 Boss 材料 ×1 + 秘银锭 ×1 + 精华 ×2

* 本幕多一个 "矿脉" 节点类型可挖材料

* 够造 1 把精钢武器 + 1 次附魔

##### 第三幕（龙巢神殿）预期收获

精钢锭 2-3 / 秘银锭 2-3 / 陨铁锭 0-1 / 精华 4-5 / Boss 材料 1



* 普通战斗：掉 精钢锭 (60%) 或 秘银锭 (40%)

* 精英战斗：掉 秘银锭 ×1 + 精华 ×2，小概率掉陨铁锭

* Boss：掉 Boss 材料 ×1 + 陨铁锭 ×1 + 精华 ×3

##### 整局总收入期望

铁锭 6-8 / 精钢锭 3-5 / 秘银锭 2-3 / 陨铁锭 0-1 / 精华 6-8 / Boss 材料 3。最多造 2-3 把武器 + 2-3 次附魔 + 1-2 次护甲升级，必须在三者之间分配。

#### 掉落决策设计



* 战后二选一： 战斗胜利后选择 \[卡牌奖励三选一] 或 \[材料奖励]，而非两者都给。前期更需要卡组 build，中后期更需要材料锻造，选择本身就是策略。

* 精英绑定材料类型： 不同精英掉不同材料，路线图上可预见。选路线 = 选材料 = 规划 build。

* 材料应急使用： 每种材料可在战斗中消耗，提供临时效果，制造 "保命还是留着锻造" 的紧张抉择：



| 材料       | 应急效果             |
| -------- | ---------------- |
| 🪨 铁锭    | 获得 8 护甲          |
| ⚙️ 精钢锭   | 获得 12 护甲         |
| 💎 秘银锭   | 本回合所有卡费用 -1      |
| ☄️ 陨铁锭   | 本回合所有卡伤害翻倍       |
| 🔥 元素精华  | 对全体敌人施加 2 灼烧     |
| ⚔️ 战魂精华  | 本场战斗力量 +2        |
| 🛡️ 守护精华 | 本场战斗每回合自动获得 3 护甲 |

#### 防极端体验

##### 保底（防一无所获）



* 每幕至少 1 个铁匠 / 篝火节点

* 每幕第一场精英必掉当幕对应材质

* 商店永远卖基础材料（铁锭 / 精钢锭）

* 最差情况也能在第一幕末造出一把铁武器

##### 上限（防材料溢出）



* 每种材质最多持有 5 个

* 精华总量最多持有 8 个

* Boss 材料专属槽位不占背包，每种最多 1 个

* 溢出时可交给铁匠换少量金币

* 不走锻造路线时材料可在商店变现买卡

## 七、敌人设计

### 普通敌人



* 地精散兵 \[HP20]：攻击 6→攻击 6→召唤地精，轮转

* 森林狼 \[HP15]：首回合嚎叫（全体友方 + 2 力量），之后攻击 8

* 蘑菇怪 \[HP25]：释放孢子（2 中毒）→攻击 5，轮转

### 精英敌人



* 暗影刺客 \[HP50]：被动 "≤5 伤害闪避"，潜行→暗杀 20→撒毒 6 中毒；逼你打大伤害

* 石像鬼 \[HP70+10 甲]：被动每回合 + 10 甲，重拳 15→石化凝视（体力 - 1）→防御 + 20 甲；需要破甲 / 魔法持续输出

### Boss（第一幕示例）



* 地精王 \[HP100]：被动 "每个地精 + 3 攻击力"

* 阶段 1（>50% HP）：召唤→重击 12→战吼

* 阶段 2（≤50% HP）：不召唤，每回合双击，攻击力 = 12 + 地精数 ×3

* 策略抉择：先清小怪还是集火 Boss

## 八、Meta 进度系统

每次冒险获得 \*\*"冒险之证"\*\*，用于永久解锁：



* 新卡牌加入地牢卡池

* 新武器蓝图加入铁匠锻造表

* 新附魔类型加入附魔池

* 新角色起始武器选项

* 挑战模式（高难度 + 特殊规则）

**设计核心**：死亡也有意义 ——"虽然死了，但解锁了毒系卡池"。
