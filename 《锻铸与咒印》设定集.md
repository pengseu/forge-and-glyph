《锻铸与咒印》(Forge & Glyph) — 游戏设计文档 v2.1
一、项目概述
类型： Web端 Roguelike 卡牌构筑游戏（HTML + CSS + JS 单页面）

世界观： 剑与魔法——地精、巨兽、巨龙、武器锻造与附魔

核心一句话： 双资源驱动的卡牌战斗 × 局内武器锻造与附魔 × 三幕式地牢探索

参考游戏： Slay the Spire（卡牌构筑）+ Hades（Meta 进度）+ Into the Breach（信息透明策略感）

与 Slay the Spire 的核心差异
Slay the Spire	锻铸与咒印
单一能量资源	体力 + 魔力双资源博弈
卡牌是唯一构筑维度	卡牌 + 武器 + 附魔三层构筑
遗物被动触发	武器主动影响卡牌效果
角色决定卡池	武器类型决定战斗风格
遗物不可锻造	武器可锻造/升级/附魔/传承
目标时长
玩家类型	单幕	全三幕
新手（首次）	10-12 / 14-18 / 16-20 min	40-50 min
熟练（5次+）	4-5 / 7-9 / 8-10 min	19-24 min
胜负条件
胜利：击败第三幕最终 Boss 并完成奖励结算
失败：战斗中玩家 HP 归零
死亡惩罚：当局一切清零（卡组/武器/金币/材料），仅保留 Meta 解锁进度
二、核心战斗系统
2.1 双资源：体力 & 魔力
每回合刷新：体力恢复至上限（默认 3），魔力恢复至上限（默认 2）
每回合抽 5 张牌
手牌在回合结束时全部弃掉
抽牌堆为空时，将弃牌堆洗回抽牌堆
体力驱动物理系卡牌（战技卡）
魔力驱动魔法系卡牌（法术卡）
技艺卡消耗任意类型或特殊费用
每回合必须在两种资源之间做取舍
2.2 三类卡牌
类别	消耗	代表
⚔️ 战技卡	体力	挥砍、格挡、重斩、处决、旋风斩
✨ 法术卡	魔力	火花、火球术、寒冰箭、陨石术、灵魂虹吸
🔧 技艺卡	任意/特殊	冥想、战吼、蓄力、磨剑、肾上腺素
稀有度分五级：⚪ basic → 🟢 common → 🔵 rare → 🟣 epic → 🟡 legendary（Act 3 引入）

2.3 回合时序（核心规则）
回合开始（startTurn）：

刷新体力/魔力至上限
护甲清算为 min(当前护甲, 屏障值)
守护精华的"每回合护甲"在此时加上
石像鬼被动在此时触发
玩家中毒在此时结算（不衰减）
抽 5 张，进入玩家行动
玩家行动：

出牌、普攻（每回合限 1 次）、使用材料、结束回合
敌方行动（endPlayerTurn）：

每个存活敌人按意图行动（冻结跳过本次行动）
回合末结算：

敌人中毒结算（不衰减）
敌人灼烧结算（结算后 -1）
虚弱/易伤层数衰减
魔法吸收回蓝判定
回合末自伤（如魔力涌流）
2.4 伤害/修正顺序
基础伤害起点（卡牌效果或武器普攻）
战技加成（combatDamageBonus）
力量加成（战技/普攻）或智慧加成（法术）
祝福附魔 +N（攻击类）
圣火共鸣 ×1.5（对有灼烧目标）
虚空附魔无视 N 护甲
铁弓首个攻击 ×1.3
铁杖法术 ×1.2
蓄能（下个法术每层 +10%，用后清零）
玩家虚弱 ×0.75
敌方易伤 ×1.5
护甲与 HP 结算
关键 UI 规则：鼠标悬停/点击卡牌时，目标敌人头上实时显示经过全部修正链计算后的预判最终伤害数字。护甲吸收部分用灰色显示，穿透部分用红色显示。此功能在 Phase 1 即实现。

2.5 状态效果
效果	说明	衰减规则
🔥 灼烧	回合末受 N 伤害	每回合 -1
🧊 冻结	跳过下一次行动，然后获得短暂免疫	触发后消除
🐍 中毒	回合开始受 N 伤害	战斗内不衰减，战斗结束后清除，上限 15 层
💪 力量	战技/普攻伤害 +N	不衰减
🔮 智慧	法术伤害 +N	不衰减
🛡️ 护甲	抵消等量伤害	回合开始清算至屏障值
🔵 屏障	护甲回合开始最多保留 N 层	不衰减
⚡ 蓄能	下个法术每层 +10% 伤害	使用后清零
💀 易伤	受到伤害 +50%	每回合 -1
😵 虚弱	造成伤害 -25%	每回合 -1
🌿 荆棘	被攻击时反伤 N	战斗内不衰减
2.6 消耗（Exhaust）关键词
标记为"消耗"的卡牌打出后从本场战斗中永久移除，不回到弃牌堆，不会被重新抽到。战斗结束后回到卡组。

2.7 敌人意图
玩家可以看到每个敌人下一回合的意图（攻击/防御/buff/特殊）
意图在回合开始时确定并显示
这是核心策略信息源
2.8 信息展示规则（Phase 1 必做）
交互	显示内容
悬停卡牌	目标敌人头上显示预判最终伤害（含全部修正）
悬停敌人	显示完整意图 + 当前所有状态效果及层数
悬停关键词	弹出简短解释（灼烧/中毒/易伤等）
手牌区	实时显示当前体力/魔力余量
抽牌堆/弃牌堆	显示剩余张数（不可查看具体内容）

四、武器系统
4.1 通用规则
仅可装备 1 把武器
装备后获得普攻（0 费），每回合限 1 次
普攻算"攻击命中"类触发来源（附魔/力量等可生效）
武器可通过锻造、升级、附魔持续成长

十二、传承系统
12.1 规则
条件： 击败最终 Boss（三幕通关）后可选 1 件武器传承
死亡不可传承，只有通关才行
传承 3 次以上的武器获得"传说"前缀并记录历史战绩
12.2 传承武器弱化公式
参数	弱化规则
品质	统一降为精钢级（无论原武器是传说/秘银/精钢）
基础攻击力	原版 × 70%（向下取整）
百分比被动	降 10%（如法术 +40% → +30%）
数值被动	降 1（如击碎 5 甲 → 4 甲）
条件被动	不变（如"首张 ≤1 费战技抽 1"保持原样）
附魔	保留 1 个（传承时选择保留哪个）
持有深渊之心时	可保留 2 个附魔
12.3 传承武器出现时机
Act 2 第一个事件节点（固定触发传承事件，替代正常事件）
传承事件内容：
选项 A：装备传承武器（替换当前武器）
选项 B：分解传承武器，获得精钢锭 ×2 + 精华 ×1
12.4 与 Boss 传奇武器的关系
传奇原版可传承（弱化后下局出现）
蓝图凡人版也可传承
蓝图精通等级 3 要求用凡人版完成传承
十三、Meta 进度系统
每次冒险结束（无论胜败）获得冒险之证，用于永久解锁：

解锁类型	示例
新卡牌加入地牢卡池	"首次用中毒击杀精英 → 解锁蚀骨毒"
新武器蓝图	"首次用长剑击败地精王 → 解锁精钢长剑蓝图"
新附魔类型	"首次用烈焰附魔击杀冰系怪 → 解锁焚尽附魔谱"
新起始选项	"通关 3 次 → 解锁匕首/战锤作为起始武器选项"
挑战模式	"通关 1 次 → 解锁高难度模式"
死亡也有意义——"虽然死了，但解锁了毒系卡池"。

新手铁匠处只有 3 种基础武器，老玩家同一个铁匠处可能有几十种选择——数值差距小，路线选择丰富。
